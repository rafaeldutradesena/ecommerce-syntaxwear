# SyntaxWear - Tênis e Sneakers Online 👟

A **SyntaxWear** é uma landing page de e-commerce moderna e responsiva dedicada à venda de tênis e sneakers. O projeto foca em uma experiência de usuário limpa, destacando coleções exclusivas e categorias de produtos com um design urbano e tecnológico.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
  - **Variáveis CSS**: Para facilitar a manutenção de cores e fontes.
  - **Flexbox e CSS Grid**: Para layouts complexos e responsivos.
  - **Arquitetura Modular**: Organização dos estilos por componentes (Header, Hero, Footer, etc).
  - **Design Responsivo**: Adaptado para dispositivos móveis e desktop.

## 🎨 Funcionalidades

- **Menu Mobile**: Navegação adaptável com ícone de hambúrguer para dispositivos menores.
- **Seção Hero**: Destaque principal com chamadas para ação (CTAs) claras.
- **Categorias de Produtos**: Navegação rápida entre os estilos Casual, Moderno, Esporte e Futurista.
- **Grade de Produtos (Grid)**: Layout dinâmico para exibição dos modelos em destaque.
- **Rodapé Completo**: Inclui formulário de newsletter, links de navegação e integração com redes sociais.

## 📂 Estrutura do Projeto

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/       # Estilos específicos por seção (header, footer, etc)
│   ├── base.css          # Estilos base e globais
│   ├── reset.css         # Reset de estilos padrão do navegador
│   ├── variables.css     # Definição de cores, fontes e espaçamentos
│   └── layout.css        # Estrutura principal das seções
├── images/
│   ├── banners/          # Imagens de fundo e banners principais
│   ├── icons/            # Ícones das redes sociais e utilitários
│   ├── logo/             # Identidade visual da marca
│   └── products/         # Fotos dos tênis e modelos
├── index.html            # Arquivo principal do site
└── README.md             # Documentação do projeto
```

## 🛠️ Como Executar o Projeto

Como este é um projeto estático (HTML e CSS), você pode visualizá-lo de forma simples:

1. Clone este repositório ou baixe os arquivos.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

*Dica: Se estiver usando o VS Code, recomendo a extensão **Live Server** para visualizar as alterações em tempo real.*
